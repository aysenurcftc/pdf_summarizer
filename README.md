## Run Locally

Clone the project

```bash
  git clone https://github.com/aysenurcftc/pdf_summarizer.git
```

Go to the project directory

```bash
  cd pdf_summarizer
```

Install dependencies

```bash
    pipenv install
```

Start the server

```bash
   streamlit run app.py  
```
